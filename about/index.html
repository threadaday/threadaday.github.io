What is Thread A Day?

Thread A Day is a site that aims to provide meaningful discussion around one link or topic each day.

What a great idea! I'm sure you came up with it yourself.

Nope! It was inspired by this post from Hacker News.

Who runs this site?

This site is run by Duncan Smith, a developer & designer.

Can I submit posts?

Sure, just make a pull request: https://github.com/threadaday/threadaday

What powers Thread A Day?

Thread A Day is proudly powered by Jekyll. This decision was reached after quickly concluding that I can support HN-level traffic for free with GH Pages.
